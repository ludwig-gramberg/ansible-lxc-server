Starting point: Fresh Debian/Ubuntu server. Nothing there, only root account and password.

- New folder for server with inventory.ini:
x.x.x.x # ip of server

- Test (in folder):
$ ansible all -i inventory/hosts -m ping -u root --ask-pass

- Generate private/public key for this server (here linode):
$ ssh-keygen
Generating public/private rsa key pair.
Enter file in which to save the key (/Users/username/.ssh/id_rsa): /Users/username/.ssh/id_rsa_linode
...

- Edit Playbook. Generate password for admin user:
$ openssl passwd -salt MySalt -1 MyPassword
$1$MySalt$DW67zPSAQlpHwkG6uhjCj.

- ansible-playbook initial-setup.yml -i inventory.ini -u root --ask-pass

- Login with
  ssh -i ~/.ssh/id_rsa_linode admin@x.x.x.x

  or, in ~/.ssh/config

Host linode
 IdentityFile ~/.ssh/id_rsa_linode
 HostName x.x.x.x
 User admin

 and 
 $ ssh linode
